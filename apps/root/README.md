# Mean Finance FE

This repository contains the code for the Mean Finance public web dApp.

## 👨‍💻 Development environment

- Install dependencies

```bash
yarn install
```

- Run locally on port 3000

```bash
yarn start
```

## 🧪 Linting

```bash
yarn lint
```

Will run linter under [src](./src)
